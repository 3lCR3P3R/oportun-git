1. Clone

Permite clonar un repositorio remoto a un workspace.

git clone [path]

Ejemplo:

git clone https://github.com/jhonnyesquivel/git-basics.git

5. Pull

Obtiene los cambios realizados en el repositorio remoto para la rama especificada

git pull

Ejemplo:

7. Merge

Combina el contendio diferente entre dos ramas, los cambios seran registrados como un commit en el log.

git merge <branch>

Ejemplo:

git merge feature/pull

9. Reset

Permite retrocerder hasta un commit-id específico, requerirá ejecutar el comando git push --force

git reset --hard <commit-id>

Ejemplo:

git reset --hard 3c3ef8c55539e1eb2f0a7d606f7afca3945dfcfd
